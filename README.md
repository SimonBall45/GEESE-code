# GEESE Project – Galaxy Evolution & Cosmology Analysis

This repository contains Python scripts and datasets used for analysing galaxy properties and cosmological relationships.

## 📌 Overview

The project explores key astrophysical concepts using observational and simulated data, including:

- Stellar mass distributions  
- Star formation rates (SFR and sSFR)  
- Metallicity trends  
- Hubble diagram calculations  
- EUCLID and DES survey data analysis  


---

## 📊 Data

The project uses datasets derived from:

- DES (Dark Energy Survey)  
- EUCLID survey simulations  

Data files include:

- DES-Dovekie_Metadata.csv ---> Main dataset for DES 
- DES-Dovekie_Metadata_2.csv ---> Reformatted dataset used in DES_LOGMASS, DES_LOGSFR and DES_LOGsSFR 
- DES-EUCLID_csv ---> Combined dataset with cross-referenced EUCLID data

## 🧠 Notes

- Some scripts depend on specific CSV files — ensure file paths are correct.
